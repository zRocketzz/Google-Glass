# Google Glass

Projeto web inspirado no conceito do Google Glass, desenvolvido com foco em estruturação de páginas, estilização visual e interatividade utilizando HTML, CSS e JavaScript.

O site apresenta informações, imagens, vídeos e curiosidades sobre a tecnologia Google Glass através de uma interface dinâmica e organizada.

---

## 🌐 Visão Geral

O projeto simula um site informativo sobre o Google Glass, trazendo uma experiência visual inspirada em páginas de tecnologia e produtos digitais.

Entre os principais recursos do site estão:

- Navegação entre múltiplas páginas
- Menu interativo
- Alteração dinâmica de imagens
- Galeria de fotos
- Integração de vídeos
- Estilização personalizada
- Layout organizado e intuitivo

---

## 📄 Estrutura das páginas

### 🏠 Página Inicial
Apresenta o conceito do Google Glass, além de imagens e informações introdutórias sobre o produto.

### 📷 Fotos
Galeria com imagens relacionadas ao Google Glass utilizando efeitos visuais e estilização personalizada.

### 🎥 Multimídia
Página com vídeos e conteúdos em mídia integrados ao site.

### 📰 Especificações
Exibe detalhes técnicos do produto com organização visual e interatividade.

### 📞 Fale Conosco
Página de contato contendo formulário estilizado e elementos interativos.
